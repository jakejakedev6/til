# TIL — Today I Learned

学習ログをまとめたリポジトリです。  
A repository for logging daily learning notes.

## 概要 / Overview

毎日学んだことを短いメモとしてまとめていきます。  
Bite-sized notes on things learned day by day.

## 学習目的 / Purpose

- プログラミング・CS基礎の定着
- アウトプットによる理解の深化
- 学習の継続と記録

## 使用技術 / Tech Stack

- Markdown
- Git / GitHub
- Python（予定）
- Java（予定）

## 構成 / Structure

```
til/
├── python/
├── java/
├── git/
└── misc/
```

## 実行方法 / Usage

各ディレクトリ内の `.md` ファイルを読むだけです。  
Just read the `.md` files in each directory.
